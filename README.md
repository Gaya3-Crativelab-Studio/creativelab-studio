CreativeLab Studio is a premium creative agency website built with React and Vite, designed to showcase branding, UI/UX design, web development, product photography, packaging, digital marketing, and social content through a refined, modern digital experience. The site combines elegant storytelling, smooth animations, responsive layouts, and polished portfolio interactions to reflect a luxury creative brand. It is crafted for performance, visual impact, and seamless user experience across desktop and mobile devices.

## Installation

Clone the repository:

```bash
git clone https://github.com/your-username/creativelab-studio.git
cd creativelab-studio
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

---

## Available Scripts

| Script            | Description                           |
| ----------------- | ------------------------------------- |
| `npm run dev`     | Starts the Vite development server    |
| `npm run build`   | Builds the production bundle          |
| `npm run preview` | Previews the production build locally |

## Deployment

This project is configured for deployment on Vercel.

1. Connect your repository to Vercel.
2. Import the project.
3. Use the default build settings.
4. Deploy and enjoy instant production hosting.

For production builds, Vercel will use:

```bash
npm run build
```
